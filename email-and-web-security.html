E-mail and Web Security

E-mail Security - #1 Threat Vector

Comunicacion principal interna y externa

La comunicación principal interna y externa es fundamental en cualquier empresa u organización. Se refiere a la entrada y salida de información, así como a la comunicación interna entre los diferentes departamentos y oficinas administrativas.



Ataques como Phishing

El phishing es una de las amenazas más comunes y peligrosas en el mundo del correo electrónico. Los atacantes utilizan técnicas de manipulación para hacer que sus correos electrónicos parezcan legítimos, engañando a las víctimas para que revelen información sensible o realicen acciones perjudiciales.



Microsoft 365

Cisco Email Threat Defense se integra completamente con Microsoft 365, proporcionando detalles sobre las intenciones y riesgos de los correos electrónicos basados en amenazas. Esta integración mejora las capacidades de seguridad nativas de Microsoft 365, añadiendo una capa adicional de defensa contra amenazas avanzadas.

  • Mejorar la Seguridad Nativa: Complementa y mejora las capacidades de seguridad de Microsoft 365 sin necesidad de cambios complejos en la configuración.
  • Gestión Simplificada: Administra la seguridad del correo electrónico desde una interfaz centralizada.


E-mail Protection

Cisco Email Threat Defense

  • Deteccion de Amenazas Avanzada: Cisco ETD utiliza la inteligencia de amenazas de Cisco Talos.
    • Phishing: Correo electrónico fraudulento que intenta obtener información.
    • Spoofing: Correos que aparentan ser de una fuente confiable pero provienen de un remitente fraudulento.
    • Ransomware: Software malicioso que cifra los archivos del usuario y exige un rescate para desbloquearlos.

  • Visibilidad Completa: Proporciona una visión integral de todos los correos electrónicos que circulan dentro y fuera de la organización.
    • Monitoreo de Mensajes Internos: Detección de amenazas que se propagan a través de correos electrónicos internos, no solo los externos.
    • Alertas en Tiempo Real: Notificaciones instantáneas sobre posibles amenazas detectadas en los correos entrantes y salientes.

  • Análisis de Amenazas:Utiliza tecnologías avanzadas de aprendizaje automático y análisis de comportamiento para identificar y mitigar amenazas.
    • Detección de Comportamiento Anómalo: Identificación de patrones de correo electrónico inusuales que podrían indicar una amenaza.
    • Evaluación Continua: Verificación constante de la seguridad de los correos electrónicos a medida que son recibidos y enviados.

  • Remediación Rápida:Arquitectura basada en API.
    • Respuesta Automatizada: Eliminar o aislar correos electrónicos maliciosos de la bandeja de entrada de los usuarios sin intervención manual.
    • Integración sin Problemas: Funciona junto a las plataformas de correo electrónico existentes, como Microsoft 365, sin interrumpir el flujo de trabajo.

Web Security

Sitios Web

  • Redireccionamiento de URLs
    • Creacion de enlaces que parecen legítimos pero que, al hacer clic, redirigen a sitios web fraudulentos.
    • bit.ly: Acortamiento de URLs hace dificil para el usuario la autenticidad del enlace antes de hacer clic en él.


SQL Injection

SQL Injection
  • Errores: Manipulacion de las consultas de una aplicación para provocar errores. Estos pueden revelar información sobre la estructura y el contenido de la base de datos
  • Errores ciegos: El proposito del ataque es el mismo por Errores pero esta vez se obtienen por medio de datos de tipo booleano.


Control de Politicas

Mala Administracion del Control de Politicas
  • Acceso No Autorizado: Usuarios no autorizados pueden obtener acceso a sistemas y datos sensibles. Esto puede ocurrir debido a permisos incorrectos, falta de autenticación adecuada o configuraciones de seguridad débiles.
  • Filtracion de Informacion: Exposición de datos sensibles a través de brechas de seguridad. Esto puede incluir información personal, financieras o de negocios.


Cisco Secure Web Appliance (SWA)

Inspección de Tráfico Web:

Proteccion contra amenazas emergentes y conocidas, incluye la detección y bloqueo de sitios web peligrosos y la verificación de sitios desconocidos antes de permitir el acceso.



Control Avanzado del Tráfico Web

Control detallado de todo el tráfico web, incluyendo contenido web dinámico como aplicaciones de redes sociales. Puede bloquear comportamientos riesgosos sin obstaculizar la productividad del usuario.



Implementación Flexible

Despliegue Físico y Virtual: Se puede implementar como dispositivo de hardware, máquina virtual o en la nube pública utilizando Amazon Web Services y Microsoft Azure.
